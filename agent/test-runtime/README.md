# Slow Test Runtime Entries

Record each slow test in its own file to avoid merge conflicts.

## File naming

- `T-<id>.md`
- Recommended format: `T-YYYYMMDD-HHMMSS-<slug>.md`

## Required fields

```
Test: <test name or file path>
Runtime: <ms>
Cause hypothesis: <short note>
Mitigation tried: <short note>
Before/After: <metrics or notes>
```
